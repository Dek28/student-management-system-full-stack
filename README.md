Student Management System

A full-stack web application for managing students, teachers, courses, classes,
attendance, exams, results, and announcements.

Frontend: React (Vite) + TailwindCSS  
Backend: Node.js + Express  
Database: MongoDB  
Authentication: JWT

Project Structure:
- student-management-frontend
- student-management-backend

How to run:
1. Start backend (port 5000)
2. Start frontend (port 5173)

Team Work:
- One person pushes the project first
- Others clone the repo
- Everyone works on their own branch
- Changes are merged via pull requests

Note:
- node_modules and .env files are ignored
- Do not push secrets to GitHub
